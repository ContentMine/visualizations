Co-occurrence Heatmap of facts

This demo uses sample code from a ContentMine-API dump to Zenodo and can be recreated with this source code. The raw data consists of a one-month data dump from the ContentMine-API, which can be downloaded from Zenodo. It has been preprocessed and the features for each visualization have been extracted, see here for the source code.

The data consists of 473.563 individual facts extracted from 28.672 publications in the range of 1.-31. May 2016 (as indicated by first publication date by EuropePMC).
